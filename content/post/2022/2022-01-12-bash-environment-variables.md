今日發現原來係windows 上 set 左 Environment Variables  
不等如 bash 可以用那個environment variables  
可能有需要在 `Git Bash` 上
解決方法十分簡單

可以用以下command 解決這個問題

```bash
export ENVIRONMENT=dev
```

Hope you find it useful
